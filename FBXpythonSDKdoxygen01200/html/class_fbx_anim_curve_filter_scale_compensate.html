詳解

This filter tries to compensate parent's scale to children's scale. This filter is used to convert scale animation curves of nodes whose transform inherit type are eInheritRrs. In the eInheritRrs mode, child objects do not inherit scaling from parent objects at all. When a parent object is scaled, the child does not scale, but translates in order to keep proportional distance between models. If you want to change the inherit type of certain nodes from eInheritRrs to eInheritRrSs, you may call this filter to compensate scale.
