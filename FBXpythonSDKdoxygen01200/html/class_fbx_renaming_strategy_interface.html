詳解

This base class is an abstract implementation of a renaming strategy for avoiding name clashes. An implementation of a reader (FbxReader) or writer (FbxWriter) class must call a concrete implementation of "FbxRenamingStrategyInterface::Rename()" every time a name is imported or exported to avoid name clashes. Any class deriving from FbxRenamingStrategyBase must implement FbxRenamingStrategyInterface::Clear(), FbxRenamingStrategyInterface::Rename(), and FbxRenamingStrategyInterface::Clone().
