Limits Functions

Minimum and maximum value limits can be associated with properties, but FBX will not verify that these limits are respected. FBX however will store and retrieve limits from files, and will assure that they are persistent in memory while the property handle object exists.

Soft minimums and maximums values are specifying a second set of limits that can be used for UI objects such as sliders. FBX will handle them the same way it does with the normal limits.
