FbxRootNodeUtility クラス

This class collects static functions for manipulating Fbx_Root nodes. Fbx_Root nodes were used to orient and scale scenes from other graphics applications. They have been replaced by the conversion routines in FbxAxisSystem and FbxSystemUnit. These methods are provided for backward compatibility only and will eventually be removed. Use the conversion routines in FbxSystemUnit and FbxAxisSystem when possible. [詳解]
