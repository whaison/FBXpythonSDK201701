14 // The scene created in this example is a textured cube, a pyramid
15 // with materials mapped on it's faces and triangle deformed by vertex cache.
16 // An animation stack displays 6 different angles of all models.
17 //
18 // The example illustrates how to:
19 // 1) create a cube in mesh
20 // 2) map textures on diffuse, ambient and emissive channel of the cube
21 // 3) create an UV set for each channel with a texture for the cube
22 // 4) create a pyramid in mesh
23 // 5) create and map materials on the diffuse channel for each faces of the pyramid
24 // 6) create a cube with our custom mesh type
25 // 7) create and map materials on the diffuse channel for our custom mesh cube
26 // 8) create and add to our custom meshed cube a User Data Layer
27 // 9) create vertex cache deformer (by default maya caches are created in 64bits (mcx extension))
28 // 10) create an animation stack
29 // 11) animate vertex
30 // 12) export a scene in a .FBX file

32 //
33 // 1. To test vertex (3 doubles) cache (default) :
34 //
