詳解

Struct to define various options that you can use to convert the system unit of a scene. The default values are: mConvertRrsNodes = true mConvertLimits = true mConvertClusters = true mConvertLightIntensity = true mConvertPhotometricLProperties = true mConvertCameraClipPlanes = true

The default configuration have been tested to give the best conversion results in the majority of the case.

注釈
Changing any of these values will have a direct impact on the whole scene behavior.
