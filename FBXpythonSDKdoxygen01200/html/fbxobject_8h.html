template<class T >
T* FbxCast ( FbxObject *  pObject)

Safe casting of FBX SDK objects into other FBX SDK class types. This cast will perform the complete test to make sure the object inherits from the requested class type. This is the equivalent of a dynamic_cast but much faster.

引数
pObjectThe object to try to cast into T type.
戻り値
A non-null pointer if the cast was successful.
