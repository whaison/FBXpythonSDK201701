Property Limits.

Property limits are provided for convenience if some applications desire to bound the range of possible values for a given type property. FBX will never apply these limits internally, however it will store and retrieve the limits values from files, and will assure that they are persistent in memory while the property exists.

Notes:

  • The limit value is truncated to the property data type.
  • These limits are meaningless for the boolean type. It is the responsibility of the calling application to implement the necessary instructions to limit the property of boolean type.
  • The SetMinLimit/SetMaxLimit methods will do nothing if SupportSetLimitsAsDoube() returns false.
