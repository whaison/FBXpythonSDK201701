A data source is a container of raw data that lacks semantic meaning so that the data can be reused within the document. To use the data, a consumer declares a connection to it with the desired semantic information.

引数

pSemanticA semantic meaning, like "INPUT", "OUTPUT" or "INTERPOLATION" in animation system.
