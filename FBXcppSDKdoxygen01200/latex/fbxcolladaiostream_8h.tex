Parse the string into an array. The source string is made up with many groups and each group contains p\+Source\+Group\+Size units separated by spaces; The destination array is also made up with many groups and each unit contains p\+Dest\+Group\+Size units. The valid unit range in each source group is \mbox{[}p\+Source\+Unit\+Offset, p\+Source\+Unit\+Offset + p\+Source\+Valid\+Unit\+Count). The valid unit range in each destination unit is \mbox{[}p\+Dest\+Unit\+Offset, p\+Dest\+Unit\+Offset + p\+Dest\+Valid\+Unit\+Count). The units in invalid range of destination is set to a default value.
