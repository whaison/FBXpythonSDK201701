FbxTransform* FbxNodeEvalState::mTransform

mTransform is used to hold the corresponding FbxTransform of the node. This FbxTransform takes all transform-related info, including pivots, offsets, pre/post rotation, rotation order, limits, etc. The evaluation is actually done through the utility functions of FbxTransform.
